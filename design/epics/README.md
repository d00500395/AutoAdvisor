# Vehicle Selection Page — Epics & User Stories

This directory contains the project epics, user stories, and tasks for the **Vehicle Selection Page** feature, derived from the [Vehicle Selection Page Requirements](../vehicle-selection-page-requirements.md).

## Epics Overview

| # | Epic | Description |
|---|------|-------------|
| 1 | [Vehicle Selector Widget](./01-vehicle-selector-widget/) | Cascading dropdowns, type-ahead, confirmation flow |
| 2 | [VIN Lookup](./02-vin-license-plate-lookup/) | VIN decode vehicle identification |
| 3 | [My Garage](./03-my-garage/) | Save, manage, and quick-select vehicles |
| 4 | [Fitment Filtering](./04-fitment-filtering/) | Catalog filtering, PDP badges, universal parts |
| 5 | [UX & Accessibility](./05-ux-accessibility/) | Mobile, error states, WCAG compliance, entry points |
| 6 | [Data & Integration](./06-data-integration/) | ACES/PIES data pipeline, API contracts, storage |
| 7 | [Marketplace Fitment Sync](./07-marketplace-fitment-sync/) | Amazon, eBay, Walmart fitment exports |

## User Story ID Convention

`VS-{epic#}-{story#}` — e.g., `VS-1-01` = Epic 1, Story 1
